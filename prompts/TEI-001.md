# Summary

Generate a diplomatic TEI transcription from printed page images while preserving document structure, typography, spelling, and punctuation.

# Purpose

This prompt provides a reusable workflow for creating a lightweight TEI transcription of printed texts. It is intended as a starting point for editorial workflows requiring basic structural markup rather than complete semantic encoding.

# Prompt

Please transcribe these pages into TEI XML markup following these rules:

1. Use proper TEI structure with `<p>` tags for paragraphs.
2. Use `<pb n="[page number]" facs="pageImages/[filename]"/>` for page breaks.
3. Use `<hi rend="italic">` for italicized text.
4. Render em dashes as `--` (double hyphen) to distinguish them from regular hyphens.
5. Use `<head>` tags for headings.
6. Preserve the exact text, including archaic spellings and punctuation.

# Parameters

## Required

| Parameter | Description |
|-----------|-------------|
| `[page number]` | Inserted into the `n` attribute of the `<pb>` element. |
| `[filename]` | Inserted into the `facs` attribute of the `<pb>` element. |

# Assumptions

- Input consists of sequential page images.
- Image filenames are known before execution.
- Images are of sufficient quality for transcription.

# Limitations

This prompt produces only basic structural markup.

It does not attempt to identify or encode:

- verse
- speakers
- quotations
- notes
- named entities
- bibliographic structures
- other specialized TEI elements

# Extensions

More specialized prompts may extend this prompt by adding:

- verse encoding
- drama encoding
- manuscript features
- semantic markup
- named entity tagging
- bibliographic encoding
